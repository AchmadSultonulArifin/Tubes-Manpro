# Project Management Plan

**Nama Proyek**: Productive Mentor

**Productive Mentor** adalah aplikasi pendamping produktivitas yang membantu pengguna
mengelola tugas harian, waktu, dan kebiasaan produktif, serta menyediakan akses mentoring
untuk pengembangan diri. Proyek ini dikembangkan sebagai bagian dari tugas akademik
dan riset aplikasi.

**Tujuan Proyek**:
- Meningkatkan produktivitas pengguna dalam mengelola tugas harian  
- Mengurangi keterlambatan tugas melalui notifikasi deadline  
- Meningkatkan manajemen waktu dan kebiasaan produktif  
- Menyediakan akses mentor untuk pengembangan diri  
- Menjadi prototipe aplikasi untuk riset akademik / tugas akhir
